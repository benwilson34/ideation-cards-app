# ideation cards

a little web-based app for practicing creativity

## development

### installation

1. install Node.js (tested with v22.14.0)
1. clone the repo
1. install dependencies: `npm ci`

### run the server

```bash
npm start
```

## deployment

the app is hosted on GitHub Pages. See [`deploy-gh-pages.sh`](./deploy-gh-pages.sh) and the `gh-pages` branch.

```bash
npm run deploy
```
